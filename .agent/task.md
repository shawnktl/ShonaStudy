# Agent Spec — Shona Language Study

## Objective

Set up a structured language study repo for Shona (chiShona) using the knowledge-base bootstrap. Populate topics covering fundamentals of the language — greetings, noun class system, verb conjugation, common vocabulary, tonal patterns, and cultural context. Create starter notes for the most important beginner topics.

## Scope

- Files/dirs the agent SHOULD touch:
  - PROJECT_SUMMARY.md (create)
  - study/topics.md (populate with Shona study topics)
  - study/schedule.md (generate initial schedule)
  - resources/sources.md (populate with learning resources)
  - notes/ (create starter notes for key topics)

- Files/dirs the agent MUST NOT touch:
  - BOOTSTRAP.md, CLAUDE.md (already from bootstrap)

## Tasks

1. Create PROJECT_SUMMARY.md — learning Shona, goals, approach
2. Populate study/topics.md with ~40-50 topics covering:
   - Greetings and basic phrases
   - Noun class system (classes 1-21, prefixes, concordial agreement)
   - Verb structure (infinitive, present, past, future, imperative)
   - Common vocabulary by theme (family, food, body, numbers, time, nature, daily life)
   - Tonal patterns and pronunciation
   - Cultural context and customs
   - Conversational phrases and idioms
3. Populate resources/sources.md with Shona learning resources
4. Create starter notes: greetings.md, noun-classes.md, verb-structure.md, vocabulary-basics.md
5. Generate initial schedule

## Constraints

- Work on branch `agent/shona-study`
- Commit with prefix `agent:`
- Use accurate Shona — this is a real language with specific grammar rules
- Include tone markings where relevant

## Acceptance Criteria

- [ ] 40+ topics covering grammar, vocabulary, and culture
- [ ] Noun class system accurately represented
- [ ] At least 4 starter note files with real Shona content
- [ ] Resources listed

## Context

Shona is a Bantu language with a noun class system, agglutinative verb morphology, and tonal distinctions. The learner has no prior Shona knowledge. Focus on practical communication first, grammar depth second.
